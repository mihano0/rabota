
Объяснение алгоритма (C++):
      Функция permute принимает строку и индексы левой и правой границы.
      Если левая граница равна правой, значит, достигнута одна из перестановок.
      В цикле происходит обмен символов и рекурсивный вызов.
      После рекурсивного вызова символы возвращаются на исходные позиции.
